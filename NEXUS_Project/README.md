# NEXUS: A Multifunctional AI Assistant

## 📌 Overview
NEXUS is an AI-powered automation assistant developed using Python and Flask. The system integrates voice interaction, browser automation, AI assistance, and desktop-level task execution into a unified platform.

## 🚀 Features
- Voice-enabled interaction
- Browser automation
- WhatsApp and Instagram integration
- AI-assisted information retrieval
- PDF audiobook reader
- System automation using PyAutoGUI
- Flask-based web interface

## 🛠️ Technologies Used
- Python
- Flask
- PyAutoGUI
- Pyttsx3
- PyPDF2
- Face Recognition
